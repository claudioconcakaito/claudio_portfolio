# Claudio's ATO Portfolio Manager

## Project Description
Claudio's ATO Portfolio Manager is a Streamlit application designed to help users manage and visualize their ATO (Authorization to Operate) portfolios efficiently. The app provides a user-friendly interface to track and analyze various aspects of ATOs, ensuring that all necessary compliance and operational requirements are met.

## Features
- **User Authentication**: Secure login to protect sensitive information.
- **Portfolio Overview**: Overview of all ATO projects in a single dashboard.
- **Data Visualization**: Graphical representation of portfolio metrics and statistics.
- **Compliance Tracking**: Monitor compliance status and deadlines for each ATO.
- **Report Generation**: Generate reports to summarize portfolio performance.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/claudioconcakaito/claudio_portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd claudio_portfolio
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## Usage Guide
- Access the app through the designated URL (provided in the terminal after running the app).
- Log in with your credentials to access your portfolio data.
- Use the navigation bar to switch between different sections of the app.
